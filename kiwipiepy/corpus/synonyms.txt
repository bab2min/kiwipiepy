# 동의어/유의어 사전
# 형식: 동의어1/품사 태그,동의어2/품사 태그,확장 방향
# 확장 방향 (T: 양방향, O: 단방향)
# 양방향 일경우 동의어1은 동의어2로 확장 그리고 동의어2는 동의어1로 확장됨
# e.g. 나는 초콜릿 아이스크림을 좋아해 -> 나는, 초콜릿, chocolate, 초콜렛, 아이스크림, icecream, 을, 좋아해
# 단방향 일경우 동의어1은 동의어2로 확장 그러나 동의어2는 동의어1로 확장되지 않음
# e.g. 나는 삼성전자에서 미니pc를 구매했다 -> 나는, 삼성, samsung, 삼성전자, 에서, 미니pc, 슬림pc, 베어본, 구매했다
# e.g. 나는 LG에서 슬림pc를 구매했다 -> 나는, LG, 엘지, 엘지전자, 에서, 슬림pc, 구매했다

삼성/NNG,samsung/NNG,삼성전자/NNG,T

미니pc/NNG,슬림pc/NNG,베어본/NNG,O
